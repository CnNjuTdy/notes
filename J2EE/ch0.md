# 课程概述

## 课程目标

介绍JavaEE  enterprise application development

- 回顾开发基础
- 学习JavaEE架构和APIs
- 熟悉重要的术语和概念
- 怎样构造，组装，部署JavaEE应用

## 教学大纲

### JavaEE架构和APIs

- Servlet
- JSP
- JDBC/JTA
- MVC
- Web应用实例
- 中间件技术（RPC/RMI）
- JMS
- JNDI
- EJB
- Java EE应用

### JavaEE企业应用构建

- SSH
  - Struts
  - Spring
  - Hibernate

## 评分标准

平时作业（20%）+大作业（30%）+期末考试（50%）
